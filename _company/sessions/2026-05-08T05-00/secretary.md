    *   **현상:** API 키와 Secret Key를 코드 내에 직접 하드코딩하는 경우, 또는 환경 변수 설정이 누락된 경우가 주요 원인입니다. [근거: 2026-05-08T01-33]

*   **🛡️ 시스템 안정성 오류 (System Stability Error):**
